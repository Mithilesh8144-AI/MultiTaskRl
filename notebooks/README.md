# Notebooks

Jupyter notebooks for experimentation and exploration.

## Available Notebooks

### cartpole_v3.ipynb
My CartPole DQN experiments and testing.
- Basic DQN implementation testing
- CartPole environment exploration
- Training and evaluation

## Usage

Open notebooks with Jupyter:
```bash
jupyter notebook notebooks/cartpole_v3.ipynb
```

Or use JupyterLab:
```bash
jupyter lab notebooks/
```

## Notes

Notebooks are for exploratory work and testing. Production experiments are in `experiments/`.
